# Rock Paper Scissors Game

## Overview

The Rock Paper Scissors Game is a Python-based interactive game where the user competes against the computer. The computer randomly selects a move, and the program determines the winner based on standard game rules.

## Features

* User versus computer gameplay
* Random computer move generation
* Score tracking system
* Multiple game rounds
* Win, lose, and draw detection

## Technologies Used

* Python 3.x
* Random Module
* Conditional Statements
* Loops

## How to Run

1. Install Python 3.x.
2. Execute the Python script.
3. Select Rock, Paper, or Scissors.
4. View the game result.

## Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock

## Learning Outcomes

* Decision-making logic
* Random number generation
* Interactive programming
* Game development fundamentals
